Teaching

2018-winter - GLIS 630: Data Mining @ McGill University [53 students]

Requirement: GLIS 617 – Information System Design, or basic knowledge of database systems.

Data mining: This course will cover association rules mining, classification analysis, cluster analysis, text mining, basic concepts and methods of frequent patterns mining, online analytical mining (OLAM), online analytical processing (OLAP), and datawarehouse architecture. The goal is to develop new data analytic skills as well as a data-oriented problem solving perspective.
